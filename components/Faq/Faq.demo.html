WAI-ARIA 1.2 Accordion Pattern · navegue acima com teclado

KeyActionWAI-ARIA Pattern
TabMove focus to next focusable elementStandard
Enter / SpaceToggle current panel✅ Pattern
↓ Arrow DownFocus next header (wraps)✅ Recommended
↑ Arrow UpFocus previous header (wraps)✅ Recommended
HomeFocus first header✅ Recommended
EndFocus last header✅ Recommended

ARIA Structure

<div role="region" aria-label="Perguntas frequentes">
  <div>
    <h3>
      <button
        aria-expanded="true|false"
        aria-controls="panel-id"
        id="header-id"
      >
        Question text
        <span aria-hidden="true">icon</span>
      </button>
    </h3>
    <div
      id="panel-id"
      role="region"
      aria-labelledby="header-id"
      hidden={!isOpen}
    >
      Answer content
    </div>
  </div>
</div>

Usage

// Composable API (children)
<Faq defaultOpen={0}>
  <Faq.Item question="Para quem é a mentoria?">
    Executivos e líderes que conduzem operações reais...
  </Faq.Item>
  <Faq.Item question="Como funciona o diagnóstico?">
    Uma conversa de 30 minutos onde mapeamos...
  </Faq.Item>
</Faq>

// Data-driven API (items prop)
<Faq items={faqs} defaultOpen={0} />

// Multiple panels open simultaneously
<Faq multiple defaultOpen={[0, 2]} />

// On dark background
<Faq className="faq--on-dark" />

// Bordered card style
<Faq className="faq--bordered" />

// Compact density
<Faq className="faq--sm" />

// Custom heading level (default h3)
<Faq headingLevel={2} />

// Controlled mode (with onChange)
<Faq
  items={faqs}
  defaultOpen={null}
  onChange={(openIndices) => console.log(openIndices)}
/>
