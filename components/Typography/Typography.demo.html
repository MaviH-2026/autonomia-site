Tones

6 tones disponíveis · aplicáveis a qualquer typo-* component

tone="default"

Texto principal. AAA contrast (15.3:1) em fundos light.

tone="secondary"

Texto secundário. AA contrast (6.1:1). Para parágrafos descritivos.

tone="muted"

Texto sutil. AA-Large only (3.8:1). Apenas para texto ≥18px ou eyebrows.

tone="accent"

Texto de acento. AA-Large em fundo light, AA em dark. Para labels com numeração.

tone="on-dark"

Texto em fundo escuro. AAA contrast (16.8:1).

tone="on-dark-soft"

Texto secundário em fundo escuro. AAA (11.8:1). 70% opacity.

Usage

// Hero display
<Display>
  Forme-se como o <em>líder orquestrador</em> da era da IA.
</Display>

// Section heading (semantic h2, visual H2)
<Heading level={2}>
  Não é um curso. É uma <em>travessia</em>.
</Heading>

// Semantic h2 styled as h3 (avoid heading skip)
<Heading level={2} visualSize={3}>
  Subseção
</Heading>

// Lede paragraph
<Lede>
  Cinco encontros 1:1 para você sair capaz de orquestrar...
</Lede>

// Body text
<Body>Texto principal</Body>
<Body tone="secondary">Texto secundário</Body>
<Body size="sm" tone="muted">Texto pequeno e sutil</Body>

// Eyebrow
<Eyebrow>Mentoria executiva · 2026</Eyebrow>
<Eyebrow tone="accent">01 · Tecnologia</Eyebrow>
<Eyebrow withDash>O método</Eyebrow>

// Quote (pull quote)
<Quote attribution="Igor Drews · Estrategista de IA" tone="on-dark">
  A tecnologia não é a estrela do show. Quem <em>conduz</em> é.
</Quote>

// Mono inline (URL, date, code)
<Mono>autonomiadigital.ia.br</Mono>
<Mono tone="accent">ABRIL 2026</Mono>

// Section header composition (replaces a-section-tag pattern)
<div className="typo-stack">
  <Eyebrow withDash>O método</Eyebrow>
  <Heading level={2}>Não é um curso. É uma <em>travessia</em>.</Heading>
  <Lede>A IA não é tendência. É infraestrutura...</Lede>
</div>
