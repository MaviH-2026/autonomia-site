🔘 Button Component

Atomic component cobrindo todas as necessidades interativas. WCAG 2.2 AA compliant. 5 variants × 3 sizes × 6 states = 90 combinações, mais modifiers on-dark e block.

Variants

Primary — Dark filled (CTA principal)

Default variant="primary"

Secondary — Light filled (CTA secundária)

Default variant="secondary"

Tertiary — Outline (ações terciárias)

Default variant="tertiary"

Accent — Orange filled (CTA destaque)

Default variant="accent" WCAG: 4.7:1 ✅ AA

Ghost — No bg (links em barras de ação)

Default variant="ghost"

Usage examples

// Hero CTA (primary com seta)
<Button variant="primary" size="md" arrow>
  Agendar diagnóstico
</Button>

// Secondary com play pill
<Button variant="secondary" play>
  Ver o trailer
</Button>

// CTA accent (orange) — agora WCAG AA
<Button variant="accent" size="lg" arrow>
  Quero atravessar
</Button>

// Como link (renderiza <a href>)
<Button as="a" href="/diagnostico" variant="primary" arrow>
  Agendar agora
</Button>

// Loading state
<Button variant="primary" loading>
  Enviando...
</Button>

// Block (full width) — para ofertas / sticky CTAs
<Button variant="primary" size="lg" block arrow>
  Quero atravessar
</Button>

// On-dark (em fundos escuros)
<Button variant="secondary" onDark>
  Ver método novamente
</Button>
