⚠️ Decisão crítica: A cor accent #E84A00 é o coração visual do DS, mas em fundos light ela só passa AA-Large (texto ≥18px). Para texto pequeno laranja em fundo creme/branco, você tem 3 opções:

Opção A (Recomendada): Use accent apenas para elementos large (CTAs, headlines com em, ícones, badges) e mantenha texto pequeno em text-primary.
Opção B: Adicione token --color-accent-deep: #A53400 para texto pequeno em fundo light. (5.6:1 AA pass)
Opção C: Mantenha #E84A00 mas force texto pequeno laranja a ter weight 600+ ou tamanho ≥18px.

Mitigação: Bordas decorativas (linhas tracejadas separando seções) são frequentemente consideradas decoração e não meaningful UI. Mas bordas em inputs/buttons interativos precisam ≥3:1.

Fix sugerido: Manter border-soft para divisores decorativos, mas usar --color-border-strong: #16140C (15:1) para inputs/forms. Ou criar --color-border-interactive: #8C7B68 (~3.2:1) para componentes interativos.
